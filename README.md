RomScanner 0.1 Alpha
====================

Description
===========

RomScanner scans your router to see if it's vulnerable to configuration disclosure.
If successful, hackers could compromise your router.

Vulnerable Routers
===================

- Billion 5200
- ZyXEL P-660HW T1 V2
- Huawei EchoLife HG520s
- TP Link W8901G
